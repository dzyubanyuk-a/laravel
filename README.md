# Paste

### О проекте

Данный проект позволяет любому пользователю хранить код в Интернете в течение определенного периода времени.



### Установка

- git clone https://github.com/dzyubanyuk-a/laravel.git
- composer install
- create database
- php artisan migrate
- php artisan key:generate

### Модули

- PHP: 8.0
- MySQL: 5.7
- Apache: 2.4
